## How to Use This Index

All context files are in `.codeatlas/context/`. Each source file has a corresponding
`.md` file describing its symbols in plain English.

- **`index.json`** — Full semantic index. Contains every symbol with plain-English descriptions.
- **`graph.json`** — File dependency graph. Shows import relationships.
- **`context/<path>.md`** — Per-file English descriptions.
